## 搭建 Python环境

Mac 和 Linux 自带 Python ，不用额外配置，开箱即用。需要注意的是 Python 分为两个主要大版本， Python2 和 Python3 。现在推荐实用的是 Python3 ，不过有很多优秀类库是用 Python2 写的。

```
➜  ~ python -V
Python 2.7.16
```

```
➜  ~ python3 -V
Python 3.7.3
```
